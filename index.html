Flexbox can create one dimensional layout an Grid can do both. When creating layout with flexbox,one can choose either row or column in a single view.The view can change in different devices but can't create layout with a combination of the two.While,grid can use two layouts at a single view. It can show row and column at the same time.So grid is used for making more complex mobile friendly layout.

Tailwind is a utility first css framework.It has more predefined classes than bootstraps. While,Bootstrap has many prebuilt ui kit or components. Tailwind is more customizable as it supports more color,spacing and styling options. Bootstrap is not much customizable from the perspective of tailwind. But it is popular as it has so many prebuilt components such carousel,cards,navbar,table that a web developer can copy and use.

CSS box model is a box that shows how a content is displayed in browsers. CSS box models shows how a content is wrapped by the custom and default styles. It mainly define the structural layout of a content and how much space it takes.The four main parts of CSS box models are margin,padding,border and content.

Semantic elements are the elements that gives browser a specific idea about it's contents.In some context, it can be said that semantic elements gives browser and searchine engines a precise idea about the different sections of a website. Some example of semantic elements are: nav,main,article,aside,footer tag.Semantic elements helps the search engine to understand the different contents of a website and it helps to organize the content website in a meaningful way.
